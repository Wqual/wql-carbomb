# wql-carbomb
This script will allow you to place an explosive on any vehicle, after which the person who gets on first will die as in the video... 

	Instalation:
 
	ox_inventory/data/items.lua


	["ied"] = {

  		label = "Car Bomb",
		  weight = 1,
		  stack = false,
		  close = true,
		  description = 'Bomba adesiva...',
	},
